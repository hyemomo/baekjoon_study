## 무방향 그래프 표현 방법

1. 인접 행렬 (2차원 배열)  
   0 1 2 3  
   0 X O X X  
   1 O X O X  
   2 X O X O  
   3 X X O X  


```python
graph = [
    [False, True, False, False],
    [True, False, True, False],
    [False, True, False, True],
    [False, False, True, False]
]
```
인접 리스트
```py
0 -> 1
1 -> 0 -> 2
2 -> 1 -> 3
3 -> 2

4 ....
```
이를 딕셔너리로 표현하면 다음과 같습니다!
```py
graph = {
    0: [1],
    1: [0, 2]
    2: [1, 3]
    3: [2]
}
```
둘 중 뭐를 쓸까? 공간을 많이 쓰느냐 시간을 많이 쓰느냐의 선택이다.

#### **시간은 얼마나 걸리나?**
만약 서로 연결되었는지 확인하기 위해서는
- 인접행렬에서는 `graph[1][2]` -> O(1)

- 인접리스트는 전부 조회해야함 -> O(n)

    -> 1: [0, 2,3,4,5] 이렇게 다 연결되어있을 수도 있기에!


#### 얼마나 공간을 쓰냐?
- 인접행렬은 O(n^2)
- 인접리스트는? O(n)
---
아래의 기준으로 선택한다. 


-> 시간이 더 중요하면 인접 행렬
-> 공간이 더 중요하면 인접 리스트
